# Delivery Performance, Delay Risk and Logistics Efficiency Analysis

## Project Overview
This project analyzes global supply chain operations to identify delivery delays, logistics bottlenecks, SLA compliance issues, and regional risk patterns.

## Tools Used
- Python
- Pandas
- NumPy
- matplotlib
- seaborne
- Streamlit


## Key Analysis
- Delivery Performance Analysis
- Delay Risk Assessment
- Shipping Mode Efficiency
- Regional Diagnostics
- SLA Compliance Analysis
- Customer Segment Impact Analysis

## Dashboard Features
- Interactive KPI Cards
- Delivery Risk Heatmaps
- Shipping Mode Comparison
- Regional Performance Analysis
- Data Explorer
